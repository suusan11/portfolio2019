## Tools
### Webpack
Compile SCSS to CSS and watch changes in HTML, SCSS and JS, after that browser will be reloaded automatically.

``webpack: 4.35.2``

``webpack-cli: 3.3.5``

``css-loader: 3.0.0``

###Animsition.js
A simple and easy jQuery plugin for css animated page transitions.

## Usage
Install modules

``$ npm install``

Run command

``$ npm run start``
